今天来一些经验之谈，有的时候总觉得很奇怪的操作，但是很有效

1.怎么打开5g开关？有单独一个软件，百度搜，MIUI可以用米客
2.怎么用附加摄像头拍照？mcpro24，可以用，拍照的话要不然试试手机自带kit，cti还是什么忘了，点击几次关于手机里面的版本号
3.怎么看历史消息？有两个办法，一个是用创建快捷方式里，找系统设置，找通知历史
还有一个是用ping这个软件，github有，从当下开始保存通知，防撤回估计也是同一个道理
4.怎么格式化u盘，给u盘换个格式？
用twrp或者澄狐，改变储存系统
5.怎么搞u盘启动器？
有这个软件，bing搜
6.gmail如何在国内收到？
qq邮箱有gamil提醒服务，qq里会同步qq邮箱，等于qq可以拿来收gmail
7.系统所用空间和其他文件怎么删？
sd maid扫一扫，储存空间分布图直观
8.github国内访问较慢或者打不开怎么办？
下载github app，随便往哪个评论区发你要进的仓库链接，点击就行。还有Copilot也能用
9.手机怎么看全景图？
米家全景相机app和vr720软件
10.手机怎么拍全景图？
谷歌街景软件，和谷歌相机都可以
11.如何只用蓝牙聊天？
bitchat是不二之选
12.如何只用wifi内网聊天？
快牙，飞鸽传书等等
13.怎么跨设备传文件？
Local send，快牙，类原生可以考虑小米快传，有模块
14.怎么拍夜视？（很黑的环境下亮一丢丢）
mcpro24，帧率最低，iso最高
15.怎么强制锁定方向？
爱玩工具箱或者强制旋转软件
16.手机怎么当服务器？
zero termux装thpony或者alist app
17.怎么看gps信号？
Gps pro或者Gps test app，能显示卫星种类，数目，方向，信号强度，卫星时间
